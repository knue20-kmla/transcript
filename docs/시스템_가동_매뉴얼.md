## 시스템 개요

### 구성 요소
- **프론트엔드**: GitHub Pages (`https://knue20-kmla.github.io/transcript/`)
- **백엔드 서버**: Flask (로컬 `http://localhost:5001`)
- **데이터**: `data/students.json`, `markdown/` 디렉토리
- **API**: Upstage Solar Pro API (PDF 파싱)

### 주요 기능
- 학생 생기부 PDF 업로드 및 자동 파싱
- 세특, 진로활동, 자율활동, 행동특성 조회
- 교사 평가 표현 자동 하이라이트
- 다크모드 지원

## 학생 추가 방법

### 1. PDF 업로드

1. 관리자 대시보드 (`http://localhost:5001/admin`) 접속
2. 학생 생기부 PDF 파일 선택
3. "업로드" 버튼 클릭

### 2. 자동 처리 과정

```
PDF 업로드
    ↓
Document Parse API (텍스트 추출)
    ↓
Solar Pro API (JSON 변환)
    ↓
students.json 저장
    ↓
마크다운 생성 (generate_markdown.py)
    ↓
Git 커밋 & 푸시
    ↓
GitHub Pages 자동 배포
```

### 3. 수동 마크다운 생성 (필요시)

```bash
cd /Users/apple/Desktop/생기부-시스템
python3 generate_markdown.py
```

### 4. Git 푸시

```bash
git add data/students.json markdown/
git commit -m "📤 [학생명] 학생 추가"
git push
```

## 문제 해결

### PDF 파싱 오류

**증상**: JSON 파싱 에러 발생

**해결 방법**:
1. `temp_response.json` 파일 확인
2. 오류 위치 주변 확인
3. 필요시 수동 수정 후 재시도

### 웹사이트에 반영 안됨

**원인**: 브라우저 캐시

**해결 방법**:
- Mac: `Cmd + Shift + R`
- Windows: `Ctrl + Shift + R`
- 시크릿 모드로 확인

### 과목 세특 누락

**원인**: 과목 매핑에 없는 과목명

**해결 방법**:
1. `generate_markdown.py` 열기
2. `subject_groups` 딕셔너리에 과목 추가
3. 마크다운 재생성

```python
'영어': [..., 'AP 영문학 I', 'AP 영문학 II']
```

### Solar API 오류

**증상**: 500 Internal Server Error

**해결 방법**:
1. 1-2분 대기 후 재시도
2. API 키 사용량 확인
3. PDF 크기 확인 (1MB 이하 권장)

## 시스템 구조

### 디렉토리 구조

```
생기부-시스템/
├── index.html              # 메인 웹페이지
├── backend_server.py       # Flask 백엔드 서버
├── upload_student.py       # PDF 파싱 로직
├── generate_markdown.py    # 마크다운 생성
├── data/
│   └── students.json       # 학생 데이터
├── markdown/               # 마크다운 파일
│   ├── [학생명]_세특_[과목].md
│   ├── [학생명]_진로활동.md
│   ├── [학생명]_자율활동.md
│   └── [학생명]_행동특성.md
├── pdfs/                   # 업로드된 PDF
└── docs/                   # 문서
```

### 데이터 구조

**students.json**:
```json
{
  "students": [
    {
      "이름": "홍길동",
      "학년": "3",
      "반": "2",
      "번호": "1",
      "계열": "인문사회계열",
      "프로필이미지URL": "",
      "성적": [...],
      "진로활동": {...},
      "자율활동": [...],
      "행동특성": [...]
    }
  ]
}
```

### API 엔드포인트

**백엔드 서버** (`http://localhost:5001`):
- `POST /upload` - PDF 업로드
- `GET /students` - 학생 목록
- `GET /health` - 헬스체크
- `GET /admin` - 관리자 대시보드

## 주요 설정

### API 키 설정

```bash
export UPSTAGE_API_KEY='your-api-key-here'
```

### GitHub 저장소

- Repository: `knue20-kmla/transcript`
- Branch: `main`
- GitHub Pages: 자동 배포

### 과목 매핑

`generate_markdown.py`의 `subject_groups` 딕셔너리에서 관리:
- 국어, 영어, 수학, 과학, 사회, 정보, 제2외국어, 교양

### 평가 하이라이트 키워드

`index.html`의 `highlightEvaluations` 함수에서 관리:
- 긍정: 뛰어남, 우수함, 훌륭함, 탁월함, 인상적, 기대됨 등
- 부정: 부족함, 미흡함, 아쉬움, 개선 필요 등
