# 주요 개발 내역

## 2026년 6월 18-19일 개발 세션

### 1. PDF 파싱 오류 처리 개선

**문제**: Solar API가 반환하는 JSON에 이스케이프되지 않은 따옴표 포함

**해결**:
- JSON 파싱 오류 시 상세 디버깅 정보 출력
- `temp_response.json` 파일로 전체 응답 저장
- 오류 위치 주변 10줄 표시
- 자동 수정 시도 (trailing commas, 따옴표 처리)
- Solar API 프롬프트에 JSON 규칙 명시

**파일**: `upload_student.py`

### 2. 진로활동, 자율활동, 행동특성 탭 추가

**구현**:
- 메인 페이지에 3개 탭 추가 (세특 옆)
- 각 탭별 독립 컴포넌트 (Career, Autonomy, Behavior)
- 마크다운 파일에서 내용 로드
- `generate_markdown.py`에 자동 생성 함수 추가

**파일**: `index.html`, `generate_markdown.py`

**내용**:
- **진로활동**: 진로와직업 과목 + 창체 진로활동
- **자율활동**: 창체 자율활동
- **행동특성**: 행동특성 및 종합의견

### 3. 디자인 개선

**헤더 및 제목**:
- 각 탭에 아이콘과 설명 추가
- 🎯 진로활동 / ✨ 자율활동 / ⭐ 행동특성

**마크다운 스타일링**:
- H2, H3 제목 스타일 개선 (그라데이션, 아이콘)
- 본문 카드형 디자인 (그라데이션 배경, 좌측 보더)
- 다크모드 지원
- 제목에서 중복 H1 제거

**파일**: `index.html` (CSS 및 컴포넌트)

### 4. 평가 표현 자동 하이라이트

**기능**:
- 교사의 주관적 평가 표현을 자동으로 하이라이트
- 긍정적 평가: 노란색 배경
- 부정적 평가: 빨간색 배경

**키워드 (엄격화 버전)**:
- 긍정: 뛰어남, 우수함, 훌륭함, 탁월함, 인상적, 기대됨, 돋보임, 눈에 띔, 높은 수준, 완성도 높음 등
- 부정: 부족함, 미흡함, 아쉬움, 개선 필요 등

**특징**:
- H1, H2, H3 제목은 하이라이트 제외
- 본문 내용만 강조
- 다크모드 지원

**파일**: `index.html` (`highlightEvaluations` 함수)

### 5. 과목 매핑 대폭 확장

**추가된 과목**:
- 영어: AP 영문학 I, II, 미국 문학 I, II
- 수학: AP 미적분학 BC-I, BC-II, BC I, BC II
- 과학: AP 환경과학
- 사회: 윤리학, 국제 관계와 국제기구
- 제2외국어: 프랑스어2, 프랑스어 독해와 작문 I, 중국어II, 중국어 독해와 작문 I

**로마자 변형 대응**:
- I, II (대문자)
- Ⅰ, Ⅱ (특수문자)
- 1, 2 (숫자)

**파일**: `generate_markdown.py`

### 6. 학생별 수정 사항

**박지유**:
- JSON 파싱 오류 수동 수정
- 세특 과목 매핑 추가
- 프로필 사진 업데이트

**전서윤**:
- 영어: 4개 → 6개 (AP 영문학 추가)
- 수학: 2개 → 4개 (AP 미적분학 BC 추가)
- 과학: AP 환경과학 추가
- 제2외국어: 프랑스어 과목 추가

**유나연**:
- 진로활동, 자율활동, 행동특성 데이터 확인
- 마크다운 정상 생성 완료

### 7. 버그 수정

**세특 탭 렌더링 오류**:
- 문제: 마크다운을 먼저 HTML로 파싱하여 `##`로 섹션 분리 실패
- 해결: 원본 마크다운으로 섹션 분리 후 개별 파싱

**경로 문제**:
- 진로활동/자율활동/행동특성 탭 404 오류
- `DATA_BASE + markdown/` → `GITHUB_BASE`로 수정

**제목 중복**:
- 마크다운 H1 제목과 컴포넌트 헤더 중복
- 마크다운 생성 시 H1 제거
- CSS로 H1 숨김 처리 추가
