# 정보

## 1학년 1학기 - 컴퓨터 시스템 일반

컴퓨터 시스템 일반: 항상 유쾌하고 유연한 태도로 수업에 임하는 자세를 보여주고 궁금한 점에 대해서도 능동적 으로 질문하여 문제점들을 해결해 나가려는 의지를 보여줌. 진법변환에 대한 명확한 이해로 친구들을 위해 예를 들어 정확하게 설명하였음. 아날로그 세상의 정보들을 컴퓨터의 이진수로 인한 디지털 정보로의 변환에 대해서 명확히 설명할 수 있으며, 보다 컴퓨터를 깊게 이해하고 적극적으로 활용하기 위해 개발자의 태도를 이해하고 적 용하고자 하는 의지를 다지게 됨.

## 1학년 1학기 - 프로그래밍기초

프로그래밍기초: 자바 프로그래밍 언어의 기초 문법을 익히고 예제를 작성하는 실습 활동과 기본 자료형 의 비트 수준 표현법에 대해 학습함. 반복 제어의 문법구조와 관련된 예제를 빠르게 이해했으며 반복 제어를 중 첩해서 해결해야 하는 확장 예제도 자연스럽게 구현하는 모습을 보였음.
