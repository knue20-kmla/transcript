# 수학

## 1학년 1학기 - 수학

(1학기)수학: 토론에 적극적으로 참여하며, 수학적 논리력을 보임. 자신의 의견을 잘 표현하며, 문제 해결 능력이 두드러짐. 곱셈 공식의 활용을 통해 계산을 할 때, 숫자를 하나의 문자로 바꾸고 적용할 곱셈 공식을 잘 찾음 반복되는 숫자는 같은 문자로 생각하고 주어진 식의 특징을 파악한 후에, 그 특징과 관련된 식의 꼴로 잘 변형 하였음.

## 1학년 2학기 - 수학

(2학기)수학: 함수의 개형에서 정의역과 공역의 조건으로부터 합성함수와 역함수의 값을 유추할 수 있으며, 일대 일대응 조건을 활용한 함수의 값을 구하는 추론의 과정을 발표함. 문제에 대한 이해를 통해 심화하고 관련하여 변형할 수 있는 문제를 분석하였으며, 완전학습을 통해 깊이 사고하는 것을 관찰할 수 있음.

## 2학년 1학기 - 수학I

수학I: 수업 중 교사의 발문에 적극적으로 대답하며 매 시간 수업에 집중하는 모습을 보임. 지수, 로그함수, 삼각함수 및 수열의 개념들을 잘 소화해내고, 관련 문제들을 해결해냄. 밑과 지수에 동시에 변수가 들어있는 함수에 로그를 씌워 그 최댓값을 정확하게 구해냄. 탄젠트를 포함한 삼각방정식을 풀고 다른 사람에게 잘 설명하였음 · 등차수열의 절댓값의 합을 구할 수 있음. 정해진 두 수 사이에 최대 몇 개의 등차수열의 수를 넣을 수 있는지 알아낼 수 있으며. 주어진 세 홀수가 등비수열을 이루지 않음을 증명함. 지수함수, 등비수열, 로그를 실생활에서 이용할 수 있는 경우를 탐구하여 수학의 유용성을 깨달았다고 함. 감염병 확산 모델을 탐구하여 확진자 증가 추세는 지수함수로, 기존 데이터에 관한 규칙은 등비수열로, 특정 감염자가 되는 데 걸리는 시간은 로그로 각각 계산해 보임. 지수함수, 로그함수가 사회 현상 분석에 도움이 된다는 사실과 어떻게 확진자 수를 예측할 수 있었는지 이해하였다고 함. 관심 분야와 수학을 연결하는 활동에서 창의적이고 통섭적인 면모가 두드러짐.

## 2학년 2학기 - 수학II

수학II : 다항함수의 미분법 단원에서 함수의 극한과 미분계수의 관계를 주제로 '수학 에세이' 를 작성함. 미분 계수의 정의를 이용해 특정 지점의 미분계수 값을 도출하는 과정에서 문제집이 제시하는 표준 해설의 한계를 비판적으로 분석한 점이 돋보임. 기존 해설이 분자나 분모를 변수로 나누어 식을 변형하는 기교에 집중했다면, 학생은 함수가 원점을 지난다는 조건에 주목하여 함수를 직접 다항식 형태로 설정하고 일차항의 계수가 곧 미분계수 값이 된다는 성질을 창의적으로 활용함. 이러한 직관적 접근을 통해 구하고자 하는 값을 미지수로 치환한 이차방정식을 유도함으로써, 복잡한 식의 변형 없이도 문제의 본질을 명확히 드러내는 대안적 풀이를 제시함. 특히 수학을 입시 도구로 바라볼 때, 시간 압박이 큰 실전 상황에서는 정형화된 공식 조작보다 본능적이고 직관적인 사고가 더 효율적일 수 있다는 점을 논리적으로 피력함. 또한, 자신의 풀이가 해당 문제의 특수성에 기인한 것임을 인지하고 적용 범위를 스스로 성찰하는 객관적인 태도를 보임. 무조건적인 암기식 풀이보다 원리에 기반한 직관적 사고를 지향하는 모습에서 뛰어난 문제 해결 잠재력을 확인함.
