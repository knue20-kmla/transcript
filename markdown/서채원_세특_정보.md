# 정보

## 1학년 2학기 - 컴퓨터 시스템 일반

컴퓨터 시스템 일반: 앞자리에 앉아 성실한 자세로 수업에 임하는 자세가 돋보이는 학생으로 수업반 전체 분위기를 긍정적으로 이끄는 데 일조함. 컴퓨터가 발전해 온 역사와 컴퓨터의 주요한 특성인 범용성과 융통성을 명확히 구분할 수 있음. 우리가 살고 있는 이 우주의 아날로그 신호를 디지털 신호로 이산적으로 바꿔야 컴퓨터에서 활용할 수 있음을 명확히 이해하고 설명할 수 있음. 다가올 미래 사회에서 더욱 적극적인 컴퓨터 사용이 필수적일 거라 예상하며 능동적으로 컴퓨터를 사용하는 방법에 대해 다양한 의견을 제시함.

## 1학년 2학기 - 프로그래밍기초

프로그래밍기초: 자바 프로그래밍 언어 기본 자료형들의 컴퓨터 내부 표현법을 학습하고, 프로그래밍 언어 문법을 활용한 프로그램 작성 실제 과정 실습을 진행함. 자바프로그래밍 언어를 만든 회사의 흥망성쇄에 대한 설명에 큰 관심을 보였으며, 프로그래밍 연습 과정에서는 관계 연산자 중 부등호와 등호가 들어가는 기호를 수학 시간에 일상적으로 사용하던 기호와 구분해야 한다는 것에 대해 흥미를 가지고 익혀 소스코드에 잘 반영하는 모습이 있었음. 컴퓨터가 2진법 체계를 이용해야만 하는 이유에 대해 정확하게 인지하고 있음.
