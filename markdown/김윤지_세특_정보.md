# 정보

## 1학년 2학기 - 프로그래밍기초

프로그래밍기초: 자바 프로그래밍 언어 기본 자료형들의 컴퓨터 내부 표현법을 학습하고, 프로그래밍 언어 문법을 활용한 프로그램 작성 실제 과정 실습을 진행함. 처음 다루는 프로그래밍 수업과 코딩 과정에서 조건 제어를 쉽게 이해하고 바로 예제를 해결하는데 적용하는 모습에서 뛰어난 학습 능력을 가진 학생임을 느낄수 있었음. 각 데이터 형의 비트 크기와 표현 능력의 한계 사이의 상관 관계를 알고 있음.

## 1학년 2학기 - 컴퓨터 시스템 일반

컴퓨터 시스템 일반: 그동안 컴퓨터를 사용하면서 피상적으로 알고 있었던 내용들을 학문적으로 접근하는 방법에 대해서 쉽지 않았지만, 최선을 다해서 공부한 것으로 자평함. 컴퓨터가 개발되고 난 이후 70여 년 동안의 역사를 비교적 자세히 설명할 수 있으며 초기 컴퓨터의 구조 등에 많은 관심을 보였음.
